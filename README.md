# OpticalFlow Implementations

A simple library with Implementations of various OpticalFlow methods using Classical Computer vision approaches.

## Dense Optical Flow Methods
* Block matching
* Horn & Shunck

### Depends on:
* stb_image headers
* Eigen3 Library
